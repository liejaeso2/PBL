자기소개

안녕하세요. 저는 웹 개발과 사용자 인터페이스에 관심이 많은 학생입니다. 아직 부족한 부분도 많지만, HTML과 CSS의 기본 구조를 이해하고 사용자가 편하게 이용할 수 있는 화면을 만드는 것을 목표로 공부하고 있습니다. 팀 프로젝트에서는 맡은 역할을 책임감 있게 수행하고, 다른 팀원들과 적극적으로 소통하는 개발자가 되고 싶습니다.

관심 기술

  • HTML과 CSS를 활용한 정적 웹페이지 제작
  • Flexbox를 이용한 반응형 레이아웃 구성
  • 사용자 경험을 고려한 UI 디자인
  • React 기반 프론트엔드 개발
